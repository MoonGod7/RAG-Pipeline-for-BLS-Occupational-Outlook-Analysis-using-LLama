# RAG using LLama, Open-Source Models from Ollama.

For creating a good RAG pipeline, one needs to take correct decisions based on their data and requirement at various steps like choosing the strategy for Chunking, choosing the embedding and LLM model, the system Prompt. So with time and practice one can become expert in taking these decisions.

Steps to create a good RAG Pipeline:

Data Ingestion
Chunking
Choosing an Embedding Model and Database
Embedding
Selection of a LLM Model
Prepare System Prompt
Try Different LLM Models
Fine Tuning System Prompt based on the Answers
Final Model
